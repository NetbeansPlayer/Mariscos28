¿Con qué plataforma se integraría “si fuera real”?

Este proyecto es 100% front-end (sin back-end). A continuación, se muestra de forma informativa con qué plataforma podría integrarse en un despliegue real.

Opción A — WordPress + WooCommerce

Tienda administrable, gran ecosistema de plugins y bajo costo inicial.

  • ✔ Integración por REST API (JSON)
  • ✔ Personalización alta (temas/plantillas)
  • ⚠ Requiere mantenimiento y actualizaciones

Opción B — Shopify

Arranque rápido con hosting incluido; menor flexibilidad avanzada.

  • ✔ Checkout nativo y seguro
  • ✔ Apps para ampliar funciones
  • ⚠ Dependencia de proveedor (lock-in)

Nota: En este curso todo se simulará en front-end (HTML, CSS, JS) sin back-end real.
